<h1 align="center">⚡ OxideTerm-Sync</h1>

### 一键安装后端

适用于使用了 systemd 的发行版（Ubuntu、Debian、CentOS...）。

```bash
bash <(curl -sL https://raw.githubusercontent.com/karami8/oxideterm-sync-server/main/install.sh)
```

## 支持同步的核心内容

- **连接配置 (Connections)**：同步所有 SSH / SFTP 的基本连接信息以及多级跳板机 (Proxy Chain) 配置。
- **安全凭据 (Credentials)**：端到端加密同步您的服务器密码与密钥解密口令 (Passphrase)。
- **端口转发 (Forwards)**：同步 Local、Remote 及 Dynamic(SOCKS5) 各类转发规则及自动启动状态。
- **全局设置 (Settings)**：漫游您的终端外观主题、操作习惯及各插件的偏好设置。
